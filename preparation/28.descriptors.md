# Дескрипторы свойств
> Дескрипторы свойств (`property descriptors`) в JavaScript - это специальные **объекты**, которые описывают атрибуты и значения свойств 
> объекта. Они предоставляют информацию о свойствах объекта, такую как его значение, возможность изменения, перечисляемость и возможность удаления.

## Основные дескрипторы свойств
1. `value`: Значение свойства.
2. `writable`: Определяет, можно ли изменять значение свойства.
3. `enumerable`: Определяет, свойство перечисляемо или нет.
4. `configurable`: Определяет, можно ли изменять атрибуты свойства и удалять его из объекта.

### Пример для Object.defineProperty()
```js
const obj = {};

Object.defineProperty(obj, 'property1', {
  value: 'value1',       // Значение свойства
  writable: true,        // Можно ли изменять значение свойства
  enumerable: true,      // Свойство перечисляемое
  configurable: true     // Можно ли изменять атрибуты свойства и удалять его
});

```

### Пример для Object.defineProperties()
Для того чтобы дать описание нескольким свойствам объекта сразу:
```js
const obj = {};

Object.defineProperties(obj, {
  property1: {
    value: 'value1',
    writable: true,
    enumerable: true,
    configurable: true
  },
  property2: {
    value: 'value2',
    writable: false,
    enumerable: false,
    configurable: false
  }
});

```
Для того чтобы применить какие-то дескрипторы сразу ко всем свойствам объекта, можно запустить цикл обхода объекта по ключам, и для 
каждого ключа передать объект дескрипторов:
```js
const obj = {
  a: 1,
  b: "Hello",
  c: true,
};

for (const key in obj) {
  Object.defineProperty(obj, key, {
    writable: false,
    enumerable: false,
    configurable: false
  });
}

// Проверка
console.log(Object.getOwnPropertyDescriptor(obj, 'a'));
console.log(Object.getOwnPropertyDescriptor(obj, 'b'));
console.log(Object.getOwnPropertyDescriptor(obj, 'c'));

```
