# Операторы сравнения
## Приведение типов "=="
Двойное равенство – сравнение с приведением ТИПОВ
```js
0 == false // true
1 == true // true
1 == "1" // true
```

### Строгое равенство "==="
Тройное равенство является СТРОГИМ, сравнивает как по типу, так и по значению.
```js
0 === false // false
1 === true // false
1 === "1" // false
```
