# Как поменять контекст функции
## `bind()`
В параметры функции передаётся контекст.  
Он возвращает новую функцию, привязанную к контексту, который был передан параметром.

```js
function foo() {
  return this;
}

const obj = { name: 'John' }
const newFoo = foo.bind(obj)

console.log(newFoo()) // { name: 'John' }
```

## `call()` and `apply()`
Они оба вызывают функцию и первым параметром принимают контекст, к которому необходимо привязать эту функцию.  

`call()` принимает параметры в виде перечисления через запятую
`appy()` принимает параметры, которые прокидывает в исходную функцию в виде массива

```js
function foo() {
  return this;
}

const obj = { name: 'John' }

foo.call( obj, 'arg1', 'arg2' ) // { name: 'John' }
foo.apply( obj, [ 'arg1', 'arg2' ] ) // { name: 'John' }
```
