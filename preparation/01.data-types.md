# Типы данных
В JS существует восемь основных типов данных:

---
1. `number` = 5
2. `string` = "I am a string"
3. `boolean` = true / false
4. `null` – значение не известно (задаётся явно)
5. `undefined` – если переменная объявлена, но значения нет
6. `Object` = {id: 123, name: 'John'}
7. `Symbol` = new Symbol("id") – уникальный идентификатор в объектах 
8. `BigInt` = 1234567890123456789n – для создания очень больших чисел
---
